# T7F-BattleSimulator
Simulatore di lotte pokémon (progetto di roleplay per il mio personaggio del **Torneo dei Sette Fiori**).

In sintesi il mio personagio diventa un allenatore e si iscrive al torneo dei sette fiori per comprendere al meglio i meccanismi delle lotte tra pokémon allo scopo di ricavarne un accurato simulatore.
Soprattutto all'inizio il programma non è inteso per essere un buon prodotto, dovrà migliorare nel tempo (si spera).
Tutto l'aiuto offerto è ben accetto quindi contattatemi per contribuire.
